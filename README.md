# introduction 

This project make for information system security project.

recommended use:

- **TypeScript v4.8.3**

- **Nodejs v16.15.1**
 
- **Npm v8.11.0**

# installation

install global package 
```
yarn global add typescript nodemon
```
or 
```
npm install -g typescript nodemon
```

install local packages 

```
yarn install 
```
or 
```
npm install
```

Then add your Config in ```.env``` file.
